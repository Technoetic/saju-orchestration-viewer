DIAGRAM 02-C

만월 아씨 사주 풀이 — 버티컬 세부 파이프라인

클라이언트 결정론 13단계 + LLM 자연어 1회 (관상과 정반대 아키텍처)
사주 결정론 (1~9 클라이언트) 서버 (myeong·LLM·crisis) Gemini 자연어 (10) ━━ 위기·면책 ━━ 응답 종착
관상 vs 만월 아씨 사주 비교:
• 관상: 서버 100% + Opus·Gemini 2단계 LLM + 21초 (LLM 93% 지연). 사진 분석이 본질.
• 만월 아씨 사주: 클라이언트 결정론 80% (절기·4주·십성·운성·신살·대운·세운) + LLM 1회 자연어 + 12초 (LLM 90% 지연). 결정론 데이터가 본질, LLM은 어조 변환만.
• 공통: ADR-014 단정 회피 + ADR-006 자문 거절 + 위기 탐지 + 면책 자동 부착.
• 최적화 차이: 관상은 Opus를 Haiku 다운그레이드 외 방법 없음. 사주는 절기·4주 계산 캐시 (생년월일 동일 시 1~9단계 통째 hit) + LLM 응답 캐시 둘 다 가능.
