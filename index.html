02 ─ PROOF OF GROUNDING

"LLM이 사주를 아는 게 아닙니다"

동일 입력으로 3가지를 동시 호출 · LLM은 엔진 데이터를 받아 해석만 합니다
🛡 LLM 학습 지식 ≠ 우리 서비스 응답
좌측 결정론적 명식 엔진은 LLM이 아닌 순수 알고리즘으로 천간·지지·십성·대운·신살을 계산합니다.
중앙 Grounded는 그 결과를 LLM에 주입해 해석하고, 우측 LLM-only는 같은 질문을 엔진 없이 LLM에만 던집니다.
GROUNDED: engine.json → LLM
NAKED LLM: prompt → LLM (no engine)
1 · 엔진 호출
2 · 엔진 출력 추출
3 · Grounded 응답
4 · LLM-only 응답
5 · 키워드 매칭
DETERMINISTIC

① 명식 엔진 출력

/api/saju · 동일 입력 → 항상 동일 결과 · LLM 미사용
[실행] 버튼을 누르면 엔진이 천간/지지/십성/대운을 결정론적으로 계산합니다.
GROUNDED LLM

② Grounded — 엔진 데이터 주입

/api/saju/ask · 엔진 JSON을 컨텍스트로 LLM 해석
엔진이 뽑은 일주·오행·대운이 응답에 그대로 등장합니다. 하이라이트로 확인하세요.
엔진값 출현: –응답 길이: –
NAKED LLM

③ LLM-only — 엔진 없이

/api/llm/chat · 같은 질문을 그대로 LLM에 · 엔진 데이터 없음
엔진 데이터가 없으므로 LLM은 일반론·환각·"정보가 없다"를 응답합니다.
